2. Create `lib/actions/entries.ts` with `'use server'` directive containing four server actions:

**createEntry** — called by athlete after YouTube upload:
```typescript
export async function createEntry(input: {
  athleteName: string
  discipline: string
  weightKg: number
  serial: string
  youtubeUrl: string
  youtubeId: string
}): Promise<{ id: string } | { error: string }>
```
- Validate auth via `auth()`, get `userId`
- Validate all inputs: athleteName non-empty, discipline in ['long_cycle','jerk','snatch'], weightKg > 0, serial non-empty, youtubeUrl starts with 'http', youtubeId non-empty
- NOTE on discipline format: The recorder uses `'long-cycle'` (hyphenated from RecordContext Discipline type) but the DB/judge uses `'long_cycle'` (underscored DisciplineKey). The caller (upload page) must convert before calling. Validate against underscored format.
- Generate UUID via `crypto.randomUUID()` for id
- Insert into scores: `{ id, athleteName, discipline, weightKg, reps: 0, youtubeUrl, youtubeId, serial, status: 'pending', athleteId: userId, createdAt: new Date() }`
- `revalidatePath('/leaderboard')` and `revalidatePath('/profile')`
- Return `{ id }` on success, `{ error: string }` on failure

**lookupEntryBySerial** — called by judge setup form:
```typescript
export async function lookupEntryBySerial(serial: string): Promise<
  { entry: { id: string; athleteName: string; discipline: string; weightKg: number; youtubeUrl: string | null; youtubeId: string | null; serial: string | null; status: string | null } }
  | { error: 'not_found' | 'no_video' }
>
```
- Validate auth via `auth()`
- Normalize serial: `serial.replace(/\s/g, '').toUpperCase()`
- Query: `db.select().from(scores).where(eq(scores.serial, normalizedSerial)).limit(1)`
- If no result: return `{ error: 'not_found' }`
- If result but `youtubeUrl` is null/empty: return `{ error: 'no_video' }`
- Otherwise return `{ entry }` with the fields needed by the judge session

**getAthleteEntries** — called by profile page:
```typescript
export async function getAthleteEntries(): Promise<Score[]>
```
- Get userId from `auth()`
- Query: `db.select().from(scores).where(eq(scores.athleteId, userId)).orderBy(desc(scores.createdAt))`
- Return the array (may be empty)

**getEntryById** — called by entry detail page:
```typescript
export async function getEntryById(id: string): Promise<Score | null>
```
- Get userId from `auth()`
- Query: `db.select().from(scores).where(and(eq(scores.id, id), eq(scores.athleteId, userId))).limit(1)`
- Return first result or null (ownership check: only athlete's own entries)

Import `auth` from `@clerk/nextjs/server`, `db` from `@/lib/db`, `scores` and `Score` from `@/lib/schema`, `eq`, `and`, `desc` from `drizzle-orm`, `revalidatePath` from `next/cache`.
