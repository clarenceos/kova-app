Create `app/registration/[compId]/page.tsx`. This page has TWO parts:
1. A **server component** wrapper that fetches competition data and determines guard state
2. A **client component** form (either inline with 'use client' in a separate section, or as a component within the same file)

**Architecture:** Since the page needs server-side data fetching for guard states (D-09, D-10) AND client-side interactivity for the form, use this pattern:
- The default export is a **server component** that fetches competition data
- It renders either a guard card OR a `<RegistrationForm>` client component
- The `<RegistrationForm>` is defined in the same file with 'use client' at the top of a separate component, OR better: the entire page file is a server component that imports a client form component

**Best approach:** Make the page.tsx a server component. Extract the form into a client component defined WITHIN the same file using the pattern: server component fetches data, renders guard or passes serialized competition data to a client form component. Since Next.js App Router requires 'use client' at the file level, create the form as a separate file OR use the pattern where page.tsx is server-side and the form is a named export from a _components directory.

**SIMPLEST CORRECT APPROACH:** Make `page.tsx` a server component. Create `app/registration/[compId]/_components/RegistrationForm.tsx` as the client component. Pass the competition data as props.

const newErrors: Record<string, string> = {}
if (!lastName.trim()) newErrors.lastName = 'Last name is required'
if (!firstName.trim()) newErrors.firstName = 'First name is required'
if (!gender) newErrors.gender = 'Gender is required'
if (!bodyWeight || parseFloat(bodyWeight) <= 0) newErrors.bodyWeight = 'Body weight is required'
if (!country) newErrors.country = 'Country is required'

const checkedEvents = Object.entries(events).filter(([, v]) => v.checked)
if (checkedEvents.length === 0) newErrors.events = 'Select at least one event'

for (const [key, val] of checkedEvents) {
  if (!val.bellWeight) {
    const eventName = key === 'LC' ? 'Long Cycle' : key === 'JERK' ? 'Jerk' : 'Snatch'
    newErrors[`${key}_bellWeight`] = `Select a bell weight for ${eventName}`
  }
  if (allowedDurations === 'both' && !val.duration) {
    const eventName = key === 'LC' ? 'Long Cycle' : key === 'JERK' ? 'Jerk' : 'Snatch'
    newErrors[`${key}_duration`] = `Select a duration for ${eventName}`
  }
}

**Submit handler:**
```typescript
async function handleSubmit(e: React.FormEvent) {
  e.preventDefault()
  setSubmitError('')
  // ... validation above ...
  if (Object.keys(newErrors).length > 0) { setErrors(newErrors); return }
  setErrors({})
  setSubmitting(true)
  try {
    const eventPayload = checkedEvents.map(([key, val]) => ({
      event: key as 'LC' | 'JERK' | 'SNATCH',
      bellWeight: val.bellWeight,
      duration: allowedDurations === 'both' ? val.duration! : parseInt(allowedDurations),
    }))
    const result = await registerAthlete({
      competitionId,
      lastName: lastName.trim(),
      firstName: firstName.trim(),
      gender: gender as 'Male' | 'Female',
      bodyWeightKg: parseFloat(bodyWeight),
      country,
      club: club.trim() || null,
      coach: coach.trim() || null,
      events: eventPayload,
    })
    if ('error' in result) {
      setSubmitError(result.error)
      return
    }
    router.push(`/registration/${competitionId}/success?registrantId=${result.registrantId}`)
  } catch {
    setSubmitError('Registration failed. Please try again.')
  } finally {
    setSubmitting(false)
  }
}
```
