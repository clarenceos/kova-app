**5b. assignJudges Pure Function (NEW lib/queue/assignJudges.ts)**

Create a pure function with zero DB imports:

```typescript
import type { TimeBlock, PlatformSlot, JudgeConflict, JudgeCandidate } from './types';

export function assignJudges(
  timeBlocks: TimeBlock[],
  candidates: JudgeCandidate[]
): { timeBlocks: TimeBlock[]; judgeConflicts: JudgeConflict[] }
```

Algorithm:
1. Build a `competingBlocks` map: `Map<registrantId, Set<blockNumber>>` — for each block, for each non-null slot, add `slot.registrantId -> block.blockNumber`. This tracks which blocks each person is competing in.

2. Build a `judgingAssignments` map: `Map<blockNumber, Set<registrantId>>` — tracks which judges are already assigned to platforms in a given block (to prevent double-assignment within the same block).

3. For each block, for each non-null platform slot:
   a. Filter candidates to eligible list:
      - MUST have `isJudging === 1 || isJudging === 2` (available to judge)
      - MUST NOT be competing in this block (check competingBlocks map)
      - MUST NOT already be assigned to judge another platform in this block (check judgingAssignments)

   b. Score each eligible candidate (lower = better):
      - `+0` base
      - `+2` if candidate.club === slot.club (same club as athlete)
      - `+3` if candidate.club === slot.club AND coach matches (same club, similar coaching)
      - `+5` if candidate's full name (firstName + lastName, case-insensitive trimmed) matches slot.coach field — this is the athlete's own coach. Flag as OWN_STUDENT.

      Actually, simplify the penalty model:
      - `+2` if `candidate.club != null && candidate.club === slot.club` (SAME_CLUB soft penalty)
      - `+5` if candidate's full name matches slot.coach (OWN_STUDENT — flag JudgeConflict with reason OWN_STUDENT)

      These are soft penalties — the candidate is still eligible, just less preferred.

   c. Sort eligible candidates by score ascending. Pick the lowest-scoring candidate.

   d. If no eligible candidates: push a JudgeConflict with reason `NO_JUDGE_AVAILABLE`, leave slot.judge = null.

   e. If picked candidate has OWN_STUDENT penalty: assign them but also push a JudgeConflict with reason `OWN_STUDENT`.

   f. If picked candidate has SAME_CLUB penalty (score >= 2 but < 5 from club match): assign them but also push JudgeConflict with reason `SAME_CLUB`.

   g. Assign: `slot.judge = { registrantId: candidate.registrantId, firstName: candidate.firstName, lastName: candidate.lastName }`

   h. Add to judgingAssignments for this block.

4. Return `{ timeBlocks, judgeConflicts }`.

NOTE: The function mutates a deep clone — caller should structuredClone before calling, OR the function clones internally. Make it clone internally for safety: `const blocks = structuredClone(timeBlocks)`.
